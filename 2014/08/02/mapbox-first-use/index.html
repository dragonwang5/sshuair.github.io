    创建自己的地图

    当然，要想创建自己的个性化地图，首先需要申请一个账号。创建完成后，在右上角可以看到Projects和Data两个选项，这里选择Projects进入可以看见自己的Projects列表，点击+ Create project进入设计地图界面。

进入设计界面后，可以看到Style、Data和Project三个主要的设计标签页，另外还有个人主页、搜索和Saved选项，下面主要介绍一下Style、Data和Project三个标签页。

Style

Style标签页主要用来设置底图的配色、底图的数据和注记语言，分别通过Color、Baselayer和Language三个选项进行设置。

  • Color：通过调色板设置不同要素（街道、建筑、绿化区、水体、陆地）的颜色。也可以通过MapBox预先定义的配色方案来设置自己地图的配色。
  • Baselayer：设置地图类型，一共三种,普通电子地图、地形图、卫星影像。
  • Language：设置主机语言，四个可选项，本地化、英语、法语、西班牙语

Data

Data标签页主要用来创建、导入和管理要素，要素类型共三种：点、线、面。

  • Browse：浏览地图以及要素和图层。
  • Marker：制作点状要素。
  • Line：制作线状要素。
  • Polygon：制作面状要素。
  • Menu：features和layers列表，管理要素和图层。

Project

Project标签页主要用来设置工程的属性以及分享权限的的控制。

  • Info：获取要素的JSON和KML格式数据；MapID（需要配合Mapbox JavaScript API使用）；分享链接和嵌入网页的链接。
  • Settings：设置工程的名称和描述。
  • Advanced：功能包括，保存当前地图位置；开关工程的公共API；forcing @2x scale for Mapbox Streets （用于支持视网膜屏幕的手机）。
