sublime text 3 配置

2014-03-05

每次重装系统时，都会重新配置一下sublime，很是麻烦记录，在此记录下自己的sublime的配置，方便以后查阅，大部分的插件可以到Package Control去下载。

一、通用插件

1.Package Control

Package Control这个自不必说，后面的插件几乎都是利用这个安装的，安装方法：Ctrl+` 打开控制台或者在菜单View→Show Console，在Console中输入一下代码：

重启sublime，OK！ 后面所有的插件都是利用这个插件进行安装。

2.Theme - Soda

Theme - Soda是一款十分漂亮的sublime主题。按Ctrl+Shift+P呼出控制栏，输入Package Control可以发现Install Package选项，选中该项，等待对话框出现输入Theme - soda回车，等待安装完成。

使用：Preferences -> Settings - User添加如下代码："theme": "Soda Dark 3.sublime-theme"

3.Monokai Extended

Monokai Extended增强Monokai Color Scheme，添加Markdown，LESS，Handlebars语法高亮，同时提升了RegEx, HTML, LESS, CSS, JavaScript高亮性能。

使用：Preferences -> Color Scheme -> User -> Monokai Extended

4.ConvertToUTF8

ConvertToUTF8支持中文编码

5.IMESupport

IMESupport使用中文输入法时，可能会出现输入法的备选字不在光标所在位置，而是在sublime的底部或顶部，那么则可以用这个插件使其恢复正常状态，跟随光标。

6.SideBarEnhancements

SideBarEnhancements侧边栏增强工具。

二、Markdown

1.Markdown Preview

Markdown Preview可以实现预览功能（在浏览器中）。

2.Markdown​Editing

Markdown​Editing为mardown编辑提供一个更好的编辑视图，支持语法高亮，支持3种格式的markdown：Standard Markdown, GitHub flavored Markdown, MultiMarkdown。并提供了三种主题：灰色、黑色和黄色。不过这个安装上后，sublime会经常出现未响应的情况。

3.右键快捷键

对于portable版本来说，没有文件右键快捷键，我们可以在注册表中添加快捷键，打开注册表在HKEY_CLASSES_ROOT -> * -> shell下面新建Sublime Text项，再在新建的Sublime Text项下面新建一个command项，将command项的数值数据设置为sublime.exe的路径，并在后面加一个%1，比如我的D:\SystemTools\Sublime Text\sublime_text.exe %1
